Strategy Two-way door or one-way door — make me decide

Bezos's frame for fast vs slow decisions. Forces a structured answer instead of stewing on it for a week.

I'm deciding whether to [DECISION].

Apply the two-way door test: is this reversible cheaply, or not?

If two-way: give me 3 fast options and tell me to pick within 24h.
If one-way: walk me through a one-page reasoning doc with risks, mitigations, and the smallest test I could run first.

Marketing Test 5 hooks before you write the post

Generates ranked hook options for the same idea so you know which one to chase before you commit a draft.

Research Steel-man the argument you disagree with

Forces Claude to write the strongest version of the opposing view before letting you respond. Kills weak takes.

I disagree with [POSITION].

Before I respond, steel-man it. Write the strongest possible version of the opposing argument:
- The 3 best reasons someone smart would hold this view
- The empirical evidence that supports it
- The 1 thing my position fails to address

Only after that, help me respond.
